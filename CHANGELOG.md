## [1.0.0] - 2026-01-28

### Added
- Initial release of Notes to My Future Self Skill
- SKILL.md with comprehensive documentation of core workflows
- README.md with overview and quick start guide
- CHAT_SUMMARY_TEMPLATE.md - blank template for creating new summary entries
- EXAMPLE_ENTRY.md - completed example showing sunscreen research decision
- PROMPT_FOR_EXTRACTING_SUMMARIES.md - extraction prompt for converting conversations to summaries
- organize_summary.py - Python helper script for validation, metadata extraction, and file organization
- MIT License for open-source sharing
- CONTRIBUTING.md for community contributions
- CHANGELOG.md for tracking version history
- Issue templates for bug reports and feature requests

### Features
- Extract summaries from long AI conversations
- Organize summaries into structured categories
- Maintain searchable personal knowledge base
- Template-based consistency for easy reference
- Python utilities for file management and validation
- Seven built-in category suggestions (Household Maintenance, Product Reviews, Health & Wellness, etc.)

## How to Use This Changelog

When making updates, add your changes under `[Unreleased]` in the appropriate section (Added, Changed, Fixed, Removed). When releasing a new version, create a new section with the version number and date.

### Types of Changes

- **Added** - New features or functionality
- **Changed** - Changes to existing functionality
- **Fixed** - Bug fixes
- **Removed** - Removed features or files
- **Deprecated** - Features that will be removed in a future version
- **Security** - Security fixes or improvements
